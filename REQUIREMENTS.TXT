SYSTEM REQUIREMENTS
-------------------
- Node.js >= 18.0.0
- npm >= 9.0.0
- .env.local file with API credentials (REQUIRED for app to run)


INSTALLATION
------------
1. Clone/extract the project
2. Run: npm install
3. Create .env.local file (copy template above)
4. Add your API credentials to .env.local
5. Run: npm run dev
6. Open browser at http://localhost:5173

ENVIRONMENT VARIABLES (.env.local)
-----------------------------------
REQUIRED: Create a .env.local file in the project root with the following variables:

VITE_SUPABASE_URL                    (Supabase project URL)
VITE_SUPABASE_ANON_KEY               (Supabase anonymous key)
VITE_FOURSQUARE_CLIENT_ID            (Foursquare API Client ID)
VITE_FOURSQUARE_CLIENT_SECRET        (Foursquare API Client Secret)
VITE_PEXELS_API_KEY                  (Pexels API key for image searches)

EXAMPLE:
--------
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
VITE_FOURSQUARE_CLIENT_ID=your-client-id
VITE_FOURSQUARE_CLIENT_SECRET=your-client-secret
VITE_PEXELS_API_KEY=your-api-key

Where to get credentials:
- Supabase: supabase.com -> Create project -> API settings
- Foursquare: foursquare.com/developers -> Create app
- Pexels: pexels.com/api -> Get API key

PROJECT STRUCTURE
-----------------
- src/               - Source code directory
- src/components/    - Reusable React components
- src/pages/         - Page components
- src/services/      - API and service integrations
- src/hooks/         - Custom React hooks
- src/lib/           - Utility libraries (Supabase)
- src/data/          - Static data files
- src/assets/        - Images and other assets
- public/            - Static files
- dist/              - Production build output (generated)
